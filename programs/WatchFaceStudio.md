# Watch Face Studio

## Useful expressions

| Meaning                      | Expression                              |
| ---------------------------- | --------------------------------------- |
| Rotate battery percentage    | `360*[BATT_PER]/100`                    |
| Rotate steps goal percentage | `360*[SC_PER]/100`                      |
| Rotate continuous seconds    | `360*[SEC_MSEC]/60` (or use sweep hand) |
